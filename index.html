سياسة الخصوصية - PrivateLessons

آخر تحديث: يونيو 2026

تطبيق PrivateLessons هو تطبيق مخصص لمساعدة الأساتذة على إدارة الأفواج التعليمية، ومتابعة حضور التلاميذ، وتنظيم الدفعات والمعلومات المتعلقة بالدروس الخصوصية.

المعلومات التي نجمعها

عند اختيار المستخدم تسجيل الدخول بواسطة البريد الإلكتروني أو حساب Google، قد نقوم بجمع المعلومات التالية:

  • عنوان البريد الإلكتروني.
  • معرف حساب المستخدم.

كما يمكن للتطبيق تخزين البيانات التي يُدخلها الأستاذ بنفسه، مثل:

  • أسماء التلاميذ.
  • الأفواج التعليمية.
  • سجلات الحضور والغياب.
  • سجلات الدفعات والمستحقات.
  • أي معلومات تعليمية أخرى يُدخلها المستخدم.

تسجيل الدخول اختياري

يمكن استخدام التطبيق دون إنشاء حساب أو تسجيل الدخول.

في حال اختار المستخدم تسجيل الدخول، تُستخدم بيانات الحساب لمزامنة البيانات وحفظها واستعادتها عند تسجيل الدخول من جهاز آخر، مما يساعد على حماية البيانات من الضياع.

كيفية استخدام المعلومات

  • التحقق من هوية المستخدم.
  • مزامنة البيانات ونسخها احتياطياً.
  • استعادة البيانات عند تسجيل الدخول من جهاز جديد.
  • توفير وظائف التطبيق الأساسية.

مشاركة البيانات

لا نقوم ببيع أو تأجير أو مشاركة بيانات المستخدمين مع أي طرف ثالث، باستثناء الخدمات التقنية الضرورية للمصادقة السحابية وتخزين البيانات ومزامنتها.

حماية البيانات

نتخذ إجراءات معقولة للمساعدة في حماية بيانات المستخدمين من الوصول غير المصرح به أو التعديل أو الإفصاح غير المصرح به.

خصوصية الأطفال

التطبيق موجّه للأساتذة ومقدمي الخدمات التعليمية. وتبقى مسؤولية إدارة بيانات التلاميذ واستخدامها على عاتق الأستاذ المستخدم للتطبيق.

التعديلات على سياسة الخصوصية

Privacy Policy - PrivateLessons

Last Updated: June 2026

PrivateLessons is an application designed to help teachers manage student groups, attendance records, payments, and other information related to private lessons.

Information We Collect

When users choose to sign in using email authentication or a Google account, we may collect:

  • Email address
  • User account identifier

The application may also store information entered by the teacher, including:

  • Student names
  • Educational groups and classes
  • Attendance records
  • Payment records
  • Other educational information entered by the user

Optional Sign-In

Users may use the application without creating an account or signing in.

If a user chooses to sign in, account information is used to synchronize and restore data across devices.

How We Use Information

  • Authenticate users
  • Synchronize and back up data
  • Restore data on new devices
  • Provide application functionality

Data Sharing

We do not sell, rent, or share user data with third parties except for services necessary for authentication, cloud storage, and synchronization.

Data Security

We take reasonable measures to help protect user information from unauthorized access, modification, or disclosure.

Children's Privacy

The application is intended for teachers and educational professionals. Any student information stored in the application is managed solely by the teacher using the application.

Changes to This Privacy Policy

We may update this Privacy Policy from time to time. Any changes will be posted on this page.
